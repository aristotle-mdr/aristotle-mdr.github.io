Editing metadata using administrator pages¶

On this page

  • Opening the editor for a metadata item

Note

This section applies to users with administrator or staff level permissions

Opening the editor for a metadata item¶

When viewing an item, click the “Actions” button in the top right of the screen to open the Actions menu, then select “Open item editor”.

Attention

If an edit link doesn’t show on your screen, you may not have permission to edit this metadata.
