Browsing the registry¶

You can browse for content within the registry.

Browsing for all types of metadata¶

The default view when browsing shows a list of all of the available metadata types within a registry, which extension they might belong to, a brief description and the number of metadata items of that type.

The first level browsing screen

Browsing for metadata by extension¶

Clicking on the name of an aristotle extension will show a list of just the items from that extension, a brief description and the number of metadata items of that type.

The browsing screen showing just standard metadata items from the Aristotle Metaedata Registry

Browsing for metadata items of a specific type¶

Clicking on the name of an aristotle extension will show a list of just the items from that extension, a brief description and the number of metadata items of that type.
